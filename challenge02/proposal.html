Challenge 02 Proposal

Purpose

Create a web app for studying the International Phonetic Alphabet.
Reference JSON or Wikipedia API to gather symbol, pronunciation, and example IPA data.
Save user progress in learning the symbols and pronounciation.
Allow user to view overall progress.

Audience

Linguistic students, or lay people interested in learning the IPA symbols and sounds.

Data Sources

localStorage for storing user progress
local JSON file or Wikipedia API for getting symbol and example data
local media files for symbol sound files

Initial Module list

- localStorage helper
- API helper?
- Card class
- Deck class
- Card Render module
- Menu Render module

Wireframes

Colors/Typography

Color Palette

Typeface

Serif for IPA symbols and menu items, sans-serif for examples, explanations, etc.
