Notes:

  • Object literal notation is the preferred way of creating objects
  • Object properties can be accessed both by dot notation and bracket notation
    - Dot notation does not work with nonstandard naming (such as spaces)
  • Ways to loop through object properties
    - 'for in' loop (includes inherited properties)
    - Object.keys() (returns the key names)
    - Object.values() (returns the values)
    - Object.entries() (returns key-value pairs)
