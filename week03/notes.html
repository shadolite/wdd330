Notes:

  • Object literal notation is the preferred way of creating objects
  • Object properties can be accessed both by dot notation and bracket notation
    - Dot notation does not work with nonstandard naming (such as spaces)
  • Ways to loop through object properties
    - 'for in' loop (includes inherited properties)
    - Object.keys() (returns the key names)
    - Object.values() (returns the values)
    - Object.entries() (returns key-value pairs)
  • Node Types
    Code Type
    1 element
    2 attribute
    3 text
    8 comment
    9 body
  • Node lists are array-like objects, but they are not arrays. You can access each item using index notation.
    For example, document.images[0] will return the first image in the node list of all the images in the document.
    To turn a node list into an array:
    - use Array.from(): 
    const imageArray = Array.from(document.images);
    - use the spread operator: 
    const imageArray = [...document.images];
  • Appending nodes:
    - Create the element node
    - Create the text node
    - Append the text node to the element node 
          const flash = document.createElement('li');
      const flashText = document.createTextNode('Flash');
      flash.appendChild(flashText);
    .insertBefore() works similarly
  • Anonymous functions cannot be removed from event listeners
  • Bubbling is when the event fires on the element clicked on
    first, then bubbles up the document tree, firing an event
    on each parent element until it reaches the root node
  • Capturing starts by firing an event on the root element,
    then propagates downwards, firing an event on each child
    element until it reaches the target element that was clicked on
